# Support

Use the right channel so reports are actionable and response times stay reasonable.

## Where to ask

- Questions / setup help: GitHub Discussions  
  `https://github.com/cobuildwithus/interface/discussions`
- Community chat: Discord  
  `https://discord.com/invite/PwWFgTck7f`
- Bug reports: GitHub Issues  
  `https://github.com/cobuildwithus/interface/issues`
- Security vulnerabilities: `SECURITY.md` / `security@justco.build`

## What issues are for

Issues are reserved for actionable, reproducible bugs and scoped feature proposals.

Please include:

- Reproduction steps
- Expected behavior
- Actual behavior
- Environment details (platform, browser, commit/version)

## Response expectations

- We do not guarantee response times.
- Security reports are prioritized.
- Issues without enough detail to reproduce may be closed.
